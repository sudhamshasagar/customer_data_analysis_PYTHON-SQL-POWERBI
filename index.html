4. Advanced SQL Techniques Showcased

Product Ranking using `RANK()`

Used a CTE and the `RANK()` window function to find the Top N products by average review rating *after* aggregation, demonstrating query efficiency and analytic function mastery (Q3).

WITH product_ranks AS (
    SELECT 
        item_purchased,
        ROUND(AVG(review_rating), 2) AS avg_rating,
        RANK() OVER (ORDER BY AVG(review_rating) DESC) AS rating_rank
    FROM customer_data GROUP BY item_purchased
)
SELECT * FROM product_ranks WHERE rating_rank <= 5;

Business Logic via `CASE` Segmentation

Implemented a critical business rule in SQL using a `CASE` statement to categorize customers based on purchase count into high-value groups (Q7).

WITH customer_type AS (
    SELECT customer_id, previous_purchases,
    CASE 
        WHEN previous_purchases = 1 THEN 'New'
        WHEN previous_purchases BETWEEN 2 AND 10 THEN 'Returning'
        ELSE 'Loyal'
    END AS customer_segment
    FROM customer_data
)
SELECT customer_segment, COUNT(*) 
FROM customer_type GROUP BY customer_segment;

5. Strategic Business Recommendations

Boost Subscriptions

Action: Promote exclusive benefits for subscribers (e.g., fast-track delivery, early sales access) to reinforce value and maximize CLV.

Tiered Loyalty Programs

Action: Reward 'Returning' buyers to successfully transition them into the high-retention 'Loyal' segment.

Review Discount Policy

Action: Shift promotions away from top-rated items to protect margins; instead, target discounts for clearance and new customer acquisition.

Strategic Product Positioning

Action: Ensure marketing and site displays prominently feature the top-rated and best-selling products identified in the SQL ranking analysis.

Targeted Marketing Focus

Action: Focus ad spend and campaigns on the **highest-revenue age groups** and customers who frequently use **Express Shipping**.
